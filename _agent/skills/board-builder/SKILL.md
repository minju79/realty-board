# Board Builder Skill

This skill automates the configuration, structure creation, code adjustments, and repository synchronization for Git-integrated static Markdown boards.

## Features
- Dynamic hybrid data model syncing local storage with GitHub remote content.
- Clean routing URLs mapping (`cleanUrls: true`).
- Markdown text converter and compiler with safe HTML element mapping.
- Dashboard with admin login state management.
